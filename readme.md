### CLI 脚手架生成项目和页面。

## 💻 安装

```
npm i antdpro-cli -g
```

## ✨ 使用教程

### 创建项目

```
antdpro-cli create project-name
```

### 创建 React 页面

```
antdpro-cli page page-name
```

## 🤖 帮助说明

```
antdpro-cli --help
```
